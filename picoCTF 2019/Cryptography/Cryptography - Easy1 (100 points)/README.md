## Walkthrough

This challenge is based on the One Time Pad (Vernam cipher). You can use this [site](http://rumkin.com/tools/cipher/otp.php) for a quick result.

In details, for each letter of the encrypted message find it in the main part of the following table, then find the line correspoding to the letter of the key with the same index. The column intersection gives you the plaintext letter.
